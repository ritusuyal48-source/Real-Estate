# 🏡 Real Estate Website

A modern and responsive Real Estate Website designed to help users explore, search, and view property listings with ease.

## ✨ Features

* Property Listings
* Property Search & Filtering
* Responsive Design
* User-Friendly Interface
* Database Integration with MySQL
* Secure Backend using PHP

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL

## 🎯 Objective

The objective of this project is to provide a simple and efficient platform for property browsing while enhancing web development and database management skills.
